## Let's Create: Chashu Pork (Easiest to Start)

### Why This One First?
- Simple circular shape
- Clear visual reference
- Good for learning shading
- Satisfying result quickly

### Setup (5 minutes)

**In Aseprite:**
```
1. New Sprite
2. Canvas: 32x32 pixels
3. Color Mode: Indexed
4. Import your palette (from earlier)
5. Layer 1: Name it "Outline"
```

### Step 1: Circle Outline (10 minutes)

```
Tools needed:
- Circle tool (hold Shift for perfect circle)
- Pencil tool (for cleanup)

Actions:
1. Select dark brown (#5C4033)
2. Draw circle: 16px diameter, centered
3. Make it 2 pixels thick
4. Manual cleanup for pixel-perfect circle
```


### Step 2: Base Fill (5 minutes)

```
1. New layer: "Base Color"
2. Move below Outline layer
3. Select pink (#E8B4A4)
4. Fill circle interior with paint bucket
```

### Step 3: Spiral Pattern (15 minutes)

```
1. New layer: "Spiral Detail"
2. Select darker pink (#C17767)
3. Draw spiral from center:
   - Start at center (1 pixel dot)
   - Curve outward clockwise
   - 2 pixels thick line
   - Make 2-3 complete rotations
   - End near edge

Reference real chashu images:
- Google "chashu pork spiral"
- Notice how fat layer creates pattern
```

**Spiral Drawing Technique:**
```
Start center ‚Üí Curve out
Keep 2-3 pixel spacing between spiral arms
Follow natural curve
Don't worry about perfection - organic is good!
```

### Step 4: Fat Line (10 minutes)

```
1. Same "Spiral Detail" layer
2. Select off-white (#FFFEF0)
3. Draw thin white line following spiral
4. Position: Between spiral and edge
5. Represents fat layer
6. 1-2 pixels thick
```

### Step 5: Shading (20 minutes)

```
1. New layer: "Shading"
2. Set to Multiply blend mode (or manually darken)
3. Light source: Top-left

Add shadows:
- Bottom-right quadrant (darker pink)
- Under spiral curves (deepest shadows)
- Edge of circle (ambient occlusion)

Add highlights:
- Top-left quadrant (lighter pink)
- On highest points of spiral
- 2-3 pixel bright spots
```

**Shading Colors:**
```
Base: #E8B4A4
Shadow 1: #D49B8C (medium shadow)
Shadow 2: #C17767 (deep shadow)
Highlight 1: #F0C4B4 (subtle highlight)
Highlight 2: #FFF0E8 (bright highlight)
```

### Step 6: Final Details (10 minutes)

```
1. New layer: "Details"
2. Add texture:
   - Tiny dots (meat texture)
   - Subtle color variation
   - Don't overdo it!

3. Edge cleanup:
   - Smooth any jagged pixels
   - Anti-alias curved edges
   - Check readability at 50% zoom
```

### Step 7: Export Test (5 minutes)

```
1. File ‚Üí Export Sprite Sheet
2. Export at 2x scale (64x64)
3. Open in Preview/Photos
4. Check clarity
5. Iterate if needed

Common issues:
- Too much detail? Simplify
- Colors too similar? Increase contrast
- Not readable small? Thicken outlines
```

### Step 8: Add Animation (15 minutes)

```
Create "Idle Bounce" animation:

Frame 1 (Base):
- Your completed pork
- Tag: "idle"
- Duration: 400ms

Frame 2 (Bounce prep):
- Squash slightly (stretch horizontal 1-2px)
- Compress vertical 1-2px
- Duration: 50ms

Frame 3 (Up):
- Stretch vertical 2-3px
- Compress horizontal 1-2px
- Shift entire sprite up 2px
- Duration: 100ms

Frame 4 (Down):
- Back to Frame 2 position
- Duration: 50ms

Export as animation strip
```

### Progression System

```swift
struct ProgressionManager {
    static func unlockIngredients(
        forLevel level: Int
    ) -> [IngredientType] {
        IngredientType.allCases.filter { 
            $0.unlockLevel <= level 
        }
    }
    
    static func nextUnlock(currentLevel: Int) -> IngredientType? {
        IngredientType.allCases
            .filter { $0.unlockLevel > currentLevel }
            .min { $0.unlockLevel < $1.unlockLevel }
    }
    
    static func progressToNextUnlock(
        currentLevel: Int
    ) -> (current: Int, needed: Int)? {
        guard let next = nextUnlock(currentLevel: currentLevel) else {
            return nil
        }
        return (currentLevel, next.unlockLevel)
    }
}
```

### Pixel Art Tips
1. **Work Small, Export Big**: Always create at 32√ó32, export at 2x/3x
2. **Consistency > Perfection**: Keep style consistent across all sprites
3. **Reference Real Food**: Look at actual ramen photos for color accuracy
4. **Test on Device**: Colors look different on phone vs monitor
5. **Animate Everything**: Even subtle movement = huge juice factor

### Development Tips
1. **Commit Daily**: Push to Git every day, even small progress
2. **Test on Device Early**: Simulator lies, test on real iPhone ASAP
3. **Profile Performance**: Use Instruments weekly
4. **Playtest with Kids**: If targeting families, test with actual kids
5. **Ship Imperfect**: Launch at 80% polish, iterate based on feedback

### Business Tips
1. **Build in Public**: Tweet progress, build community early
2. **Launch Day Matters**: Tuesday-Thursday best for iOS launches
3. **Price Right**: $2.99 sweet spot for premium puzzle games
4. **Respond to Reviews**: Engaged developer = better ratings
5. **Plan Updates**: Have v1.1 ready within 2-3 weeks of launch
