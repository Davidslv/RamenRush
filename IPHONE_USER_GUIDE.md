## ğŸ”§ Troubleshooting on iPhone

### Grid Not Responding to Touch?
1. Make sure you're tapping **inside the grid area** (center of screen)
2. Try using the **arrow buttons** instead
3. Check that the game isn't paused

### Can't See the Whole Grid?
1. Ensure your iPhone is in **vertical orientation** (upright)
2. Make sure **auto-rotate is enabled** in Control Center
3. If grid is cut off, restart the app

### Selection Not Matching Orders?
1. Check you have the **correct ingredient** (emoji matches)
2. Verify the **correct quantity** (x1, x2, or x3)
3. Make sure you selected a **line** (row or column), not diagonal

### Game Running Slow?
1. **Close other apps** - Free up iPhone memory
2. **Restart the game** - Force quit and relaunch
3. **Restart your iPhone** - Power off and on

### Lost Your Progress?
- Currently, the game resets on each launch
- Save feature coming in future update!
